# Ассеты: источники и лицензии

Весь пиксель-арт в этой папке — из экосистемы LPC (Liberated Pixel Cup)
и распространяется под свободными лицензиями с обязательным указанием
авторства. При использовании проекта сохраняйте эту папку с файлами кредитов.

## characters/ — слои персонажей

Кураторский поднабор из [Universal LPC Spritesheet Character Generator](https://github.com/LiberatedPixelCup/Universal-LPC-Spritesheet-Character-Generator)
(тела, головы, глаза, причёски, одежда, галстуки, обувь; кадры 64×64,
ходьба в 4 направления).

- Лицензии: CC-BY-SA 3.0 / GPL 3.0 (дуал), у отдельных слоёв — по данным
  из `characters/CREDITS.csv` (полный список авторов и ссылок на оригиналы).
- `characters/sheet_definitions/` — JSON-описания слоёв (z-порядок,
  совместимость, авторы) из того же репозитория.
- Недостающие категории (шляпы, платья и т.д.) можно добирать из
  апстрим-репозитория — структура папок сохранена.
