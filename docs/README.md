# Документация voffice

Документация построена по системе [Divio](https://docs.divio.com/documentation-system/)
(она же Diátaxis). Идея простая: у документации четыре разные работы, их нельзя
смешивать в одном тексте, потому что читатель приходит с разной целью.

```
                   ориентир на изучение │ ориентир на задачу
   ────────────────────────────────────┼────────────────────────────────
    практика (у клавиатуры)  Обучение   │  Инструкции
   ────────────────────────────────────┼────────────────────────────────
    теория (думаю о системе) Пояснения  │  Справочник
```

объяснить, а дать почувствовать, что всё получается.

## С чего начать

- Никогда не видели проект → [Обучение](tutorials/01-pervyy-zapusk.md).
