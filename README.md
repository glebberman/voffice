# voffice — виртуальный офис

voffice — это виртуальное пространство для команды в духе [Gather.town](https://www.gather.town/):
2D-карта офиса, по которой ходят аватары, а общение включается по близости.
Подошёл к коллеге — слышишь его в текстовом чате и видишь по видеосвязи; отошёл —
связь затухает. Есть несколько комнат с порталами между ними, интерактивные
объекты (доски, видео, карты), кастомизация персонажа и браузерный редактор карт.

Проект целиком построен на стеке **Laravel 12 + Inertia + React + Laravel Reverb + PixiJS**.

![этапы](https://img.shields.io/badge/roadmap-этапы_1–6_пройдены-2e933c)

## Что умеет

- **Движение и presence** — стрелки/WASD, коллизии, список «кто онлайн».
- **Proximity-чат** — сообщения слышны только тем, кто в радиусе 4 тайлов; в
  приватных зонах (переговорка) разговор не выходит за стены; со spotlight-плитки
  слышно всю комнату.
- **Видео/аудио по близости** — WebRTC-mesh, связь устанавливается автоматически
  с теми, кто рядом и в звонке; громкость зависит от расстояния; демонстрация экрана.
- **Реакции и статусы** — эмодзи (клавиши 1–5), статусы «доступен/занят/не
  беспокоить» + авто-«отошёл».
- **Комнаты и порталы** — несколько карт в БД, переходы между ними, лобби.
- **Интерактивные объекты** — подошёл, нажал `X` → открывается доска / YouTube /
  карта во встроенном окне.
- **Персонаж** — конструктор внешности (тело, причёска, одежда, галстук).
- **Follow / Locate / Buzz** — найти коллегу на карте, пойти за ним, позвать.
- **Редактор карт** — рисование карты в браузере (только для администраторов).

## Быстрый старт

```bash
git clone git@github.com:glebberman/voffice.git && cd voffice
composer install
cp .env.example .env && php artisan key:generate
docker compose up -d          # поднимет весь стек в контейнерах
```

Откройте <http://localhost:8000/rooms/office>. Тестовые пользователи —
`anya@voffice.test` (администратор) и `borya@voffice.test`, пароль у обоих `password`.

Подробнее — в [руководстве «Первый запуск»](docs/tutorials/01-pervyy-zapusk.md).
